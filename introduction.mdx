## Overview

Ivan Shevtsov Documentation serves as your central hub for all project documentation. You create, organize, and maintain docs in one intuitive space. This platform supports Markdown, MDX components, and version control to streamline your workflow.

## Key Benefits

You gain several advantages from this platform:

- **Collaboration**: Share docs with teams and enable real-time edits.
- **Customization**: Tailor themes and layouts to match your brand.
- **Export Options**: Generate PDFs or static sites for offline use.
- **Integrations**: Connect with GitHub, Slack, and other tools.

  Start small by creating your first page, then scale to full project suites.
</Callout>

## Get Oriented

Follow these steps to navigate and start using the platform effectively.

<Steps>
  <Step title="Explore Sidebar" icon="menu">
    Access pages, settings, and search from the left sidebar.
  </Step>
  <Step title="Create Content" icon="plus">
    Click "New Page" to add Markdown or MDX files.
  </Step>
  <Step title="Preview & Publish" icon="eye">
    Use live preview, then publish to make docs live.
  </Step>
</Steps>

<Tabs>
  <Tab title="Quick Start" icon="zap">
    Begin with the default template.

    <CodeGroup tabs="Markdown,MDX">
    ```markdown

    Welcome to your project.
    ```
    ```mdx

    Use components like `<Callout>` for richer content.
    ```
    </CodeGroup>
  </Tab>
  <Tab title="Navigation Tips" icon="compass">
